# alke-wallet
Entrega de proyecto "Wallet" correspondiente a repositorio del aplicativo, realizado por Gustavo Valverde Jarpa para el curso "Bootcamp JS Talento Digital"

alke-wallet/
├── index.html          → Entrada principal (redirección al login)
├── login.html          → Pantalla de inicio de sesión
├── menu.html           → Menú principal con resumen financiero
├── deposit.html        → Pantalla de depósito de fondos
├── sendmoney.html      → Pantalla de envío de dinero
├── transactions.html   → Historial de transacciones
├── css/
│   └── styles.css      → Estilos generales y paleta de colores
├── js/
│   └── app.js          → Lógica JavaScript y eventos jQuery
└── README.md           → Documentación del repositorio
